Description by WHO


Body Parts (Anatomical Site):  The ICD has traditionally used the topographic location of the body where the health-related problem can be found at the most specific level relevant to the condition.  This has been the starting point for assigning an ICD code. The ICD-11 should continue this approach as this definition facilitates the creation of meaningful subsets for coding and for analysis.   However, no standard anatomical site terminology or ontology has been used explicitly in creation of the ICD.  The ICD-11 Content Model requires to the allocation of an anatomical site.   Possible value sets for this characterization include: the Terminologica Anatomica;   Foundational Model of Anatomy (Either the full version or selected subset); linkage to ICF Body Structures; or SNOMED-CT body structures.
