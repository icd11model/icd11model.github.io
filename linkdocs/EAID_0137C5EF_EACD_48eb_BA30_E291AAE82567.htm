Note by Alan Rector: The important points are that:


             - Every entity in any layer should have a unique ID

             - Every entity in any layer should have a text definition. 

             - Every entity should have a presentation object.  The presentation object can be "projected" (see below) from  another layer by default, but it must be accessible   (From a programming point of view, every entity object should give  respond to Entity.presentationObject() or some similar construct.   From a query point of view, there should be a standard query to retrieve the  appropriate Presentation Object. 

             - Every entity should have at least one sort of comment, probably several.

             - The contents and further model for the presentation object needs to be worked through with
               Jon and others from the Language  community to be sure that it fits their needs. 
               If it doesn't, then we will need to extend or supplement it.
