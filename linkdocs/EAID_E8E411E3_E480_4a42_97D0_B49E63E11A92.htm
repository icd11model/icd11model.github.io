Description by WHO



The "WHO Content Model" identifies the basic characteristics needed to define any ICD category through use of multiple parameters (e.g. Body Systems, Body Parts, Signs and Symptoms, Diagnostic Findings, Causal Agents, Mechanisms, Temporal relations, Severity, Functional Impact, Treatment interventions, Diagnostic Rules).  The possible value sets need to be identified to populate this database in a relational way.  Full formal identification of this content model for each category will result in an "Information Model". 

Each ICD category should be defined along with multiple parameters as included in this content model.
