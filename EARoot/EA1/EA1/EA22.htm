<p>A type of ICD11_Entity that plays a role to hook the logical definitions from external vocabularies such as SNOMED CT, HUGO or ICF while it also plays a role for value set binding to those characteristics used to describe a clinical criteria, e.g. using "CD_Anatomical_Site" to represent the target concept descriptor of characteristic "Anatomical_Site". <br/></p><p><br/></p><p>Mapping relationship: Mappings of the concept to different coding systems (e.g. ICD-0, ICF,...) and other standardised systems like terminologies (SNOMED) etc.<br/></p>
